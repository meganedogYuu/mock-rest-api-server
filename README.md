# mock-rest-api-server

## 使い方

```bash
# clone & ライブラリインストール
$ git clone https://github.com/meganedogYuu/mock-rest-api-server.git
$ npm install

# RESTのモックサーバ起動
$ npm run sample
# => Resources
# => http://localhost:3000/movies
```
